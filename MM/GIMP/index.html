图像处理程序 GIMP

官网 https://www.gimp.org/。

安装

通过安装包安装

略。

通过 Scoop 安装

PS $env:USERPROFILE> scoop install gimp

常用操作

导出图像

GIMP Documentation 上的相关内容，及其有效中译版。
图形界面：主界面 → 菜单栏 → 文件（File） → 导出为（Export As）...
热键：Alt+F → X
快捷键：Ctrl+Shift+E

缩放图像

GIMP Documentation 上的相关内容，及其有效中译版。
图形界面：主界面 → 菜单栏 → 图像（Image） → 缩放图像（Scale Image）...
热键：Alt+I → S
