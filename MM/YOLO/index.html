    YOLO¶

    YOLO 是由 Joseph Redmon 创造的物体识别（Object Detection）算法及其改进算法。

    • YOLOv5 by Ultralytics， GitHub；
    • YOLOv4 by Alexey Bochkovskiy， GitHub；
    • YOLOv3 by Joseph Redmon， GitHub；
    • YOLOv2 by Joseph Redmon；
    • YOLOv1 by Joseph Redmon。

    参见 一文读懂YOLO V5 与 YOLO V4。

    运行 YOLOv5¶

    Window10 + CPU

    1. 下载并安装 Python；

      1. 安装完毕后，配置 pypi 镜像源：

        %USERPROFILE%> pip config set global.index-url https://pypi.doubanio.com/simple

      2. 升级 pip：

        %USERPROFILE%> python -m pip install -U pip

    2. 安装 PyTorch CPU 版本：

      %USERPROFILE%> pip install torch torchvision torchaudio

      参见这个页面；

    3. 安装 YOLOv5 依赖：

      %USERPROFILE%> pip install -r https://raw.githubusercontent.com/ultralytics/yolov5/master/requirements.txt

      提示

      如果所在地区网络受限，无法访问 https://raw.githubusercontent.com/ 域名，那么可以手动从这个页面获取该文件，保存在本地再运行：

      %USERPROFILE%> pip install -r requirements.txt

    4. 测试运行：

      import torch

# Model
model = torch.hub.load('ultralytics/yolov5', 'yolov5s', pretrained=True)

# Images
imgs = ['https://ultralytics.com/images/zidane.jpg']  # batch of images

# Inference
results = model(imgs)

# Results
results.print()
results.save()  # or .show()

results.xyxy[0]  # img1 predictions (tensor)
results.pandas().xyxy[0]  # img1 predictions (pandas)
#      xmin    ymin    xmax   ymax  confidence  class    name
# 0  749.50   43.50  1148.0  704.5    0.874023      0  person
# 1  433.50  433.50   517.5  714.5    0.687988     27     tie
# 2  114.75  195.75  1095.0  708.0    0.624512      0  person
# 3  986.00  304.00  1028.0  420.0    0.286865     27     tie

      提示

      如果所在地区网络受限，无法顺利使用 torch.hub.load 从 GitHub 加载预训练模型， 那么可以手动从这个页面获取源码包（Source code）， 解压到本地后，再从上述页面获取预训练模型（*.pt）放入解压后的项目文件夹中，并改写命令为从本地加载：

      model = torch.hub.load('path/to/local/yolov5', 'yolov5s', source='local', pretrained=True)

    运行 YOLOv3¶

    Window10 + MSYS2

    1. 下载并安装 MSYS2；

      1. 安装完毕后，通过开始菜单或 msys2 命令运行 MSYS2；

      2. 首次运行 MSYS2 将自动进行初始配置；

      3. 首次运行 MSYS2 需要手动更新包数据库并升级基础包：

        $ pacman -Syu

        升级完毕后会话将自行结束；

      4. 再次运行 MSYS2，手动更新其余基础包：

        $ pacman -Su

        现在 MSYS2 已准备完毕。

    2. 通过 MSYS2 提供 gcc 工具链支持：

      $ pacman -S --needed base-devel msys2-devel

    3. 下载 YOLOv3 源码；

      注意

      通过 Zip 压缩包形式获取源码，不要使用 git clone 命令。git clone 命令会根据操作系统适配文本文件的换行符，这将导致路径解析出问题。

    4. 进入 darknet 目录：

      $ cd darknet

    5. 编译 darknet：

      $ make

    6. 获取预训练权重文件：

      $ wget https://pjreddie.com/media/files/yolov3.weights

    7. 测试运行：

      $ ./darknet detect cfg/yolov3.cfg yolov3.weights data/dog.jpg
