# Everything

[Everything] 是一款适用于 Windows 操作系统的桌面搜索引擎。[^Everything on Wikipedia]

## 运行 Everything

### :material-microsoft-windows: Windows 10 和 Scoop

```doscon
%USERPROFILE%> scoop install everything
```
