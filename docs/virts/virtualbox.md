# VirtualBox

[VirtualBox] 是一款虚拟机软件，隶属于 [Oracle] 旗下。

## 运行 VirtualBox

### 在 :material-microsoft-windows: Windows 10 和 Scoop 环境

``` doscon
%USERPROFILE%> scoop install portable-virtualbox
```
