    Markdown¶

    Markdown 是一款轻量级标记语言，由 John Gruber 和 Aaron Swartz 在 2004 年发明。3

    Markdown¶

    标准 Markdown 支持以下语法：

    段落和折行¶

    参见 Daring Fireball: Markdown Syntax Documentation § PARAGRAPHS AND LINE BREAKS。

    Setext 风格标题¶

    Setext 风格标题示例

    This is an H1
=============

This is an H2
-------------

    渲染冲突，略。

    参见 Daring Fireball: Markdown Syntax Documentation § HEADERS。

    atx 风格标题¶

    atx 风格标题示例

    # This is an H1

## This is an H2

###### This is an H6

    渲染冲突，略。

    参见 Daring Fireball: Markdown Syntax Documentation § HEADERS。

    块级引文¶

    块级引文示例

    > This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
> consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
> Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.
> 
> Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse
> id sem consectetuer libero luctus adipiscing.

    This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet, consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus. Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.

    Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse id sem consectetuer libero luctus adipiscing.

    参见 Daring Fireball: Markdown Syntax Documentation § BLOCKQUOTES。

    无序列表¶

    无序列表示例

    *   Red
*   Green
*   Blue
    • Red
    • Green
    • Blue

    参见 Daring Fireball: Markdown Syntax Documentation § LISTS。

    有序列表¶

    有序列表示例

    1.  Bird
1.  McHale
1.  Parish
    1. Bird
    2. McHale
    3. Parish

    参见 Daring Fireball: Markdown Syntax Documentation § LISTS。

    缩进风格代码块¶

    缩进风格代码块示例

    This is a normal paragraph:

    This is a code block.

    This is a normal paragraph:

    This is a code block.

    参见 Daring Fireball: Markdown Syntax Documentation § CODE BLOCKS。

    水平线¶

    水平线示例

    * * *

***

*****

- - -

---------------------------------------

    参见 Daring Fireball: Markdown Syntax Documentation § HORIZONTAL RULES。

    内联风格链接¶

    内联风格链接示例

    This is [an example](http://example.com/ "Title") inline link.

[This link](http://example.net/) has no title attribute.

    This is an example inline link.

    This link has no title attribute.

    参见 Daring Fireball: Markdown Syntax Documentation § LINKS。

    参考风格链接¶

    参考风格链接示例

    This is [an example][id] reference-style link.

[id]: http://example.com/  "Optional Title Here"

    This is an example reference-style link.

    参见 Daring Fireball: Markdown Syntax Documentation § LINKS。

    自动链接¶

    自动链接示例

    <http://example.com/>

    http://example.com/

    参见 Daring Fireball: Markdown Syntax Documentation § AUTOMATIC LINKS。

    强调¶

    强调示例

    *single asterisks*

_single underscores_

**double asterisks**

__double underscores__

    single asterisks

    single underscores

    double asterisks

    double underscores

    参见 Daring Fireball: Markdown Syntax Documentation § EMPHASIS。

    行内代码¶

    行内代码示例

    Use the `printf()` function.

    Use the printf() function.

    参见 Daring Fireball: Markdown Syntax Documentation § CODE。

    图片¶

    图片示例

    ![Alt text](/path/to/img.jpg)

![Alt text](/path/to/img.jpg "Optional title")

    参见 Daring Fireball: Markdown Syntax Documentation § IMAGES。

    转义¶

    参见 Daring Fireball: Markdown Syntax Documentation § BACKSLASH ESCAPES。

    Python-Markdown¶

    Python-Markdown 是 Markdown 的 Python 实现，并提供了若干扩展：

    Abbreviations 扩展¶

    缩写示例

    The HTML specification
is maintained by the W3C.

*[HTML]: Hyper Text Markup Language
*[W3C]:  World Wide Web Consortium

    The HTML specification is maintained by the W3C.

    参见 Abbreviations — Python-Markdown documentation。

    Admonition 扩展¶

    告诫示例

    !!! type "optional explicit title within double quotes"
    Any number of other indented markdown elements.

    This is the second paragraph.

    optional explicit title within double quotes

    Any number of other indented markdown elements.

    This is the second paragraph.

    支持的告诫类型

    abstract, summary, tldr

    attention, caution, warning

    bug

    check, done, success

    cite, quote

    danger, error

    example

    fail, failure, missing

    faq, help, question

    hint, important, tip

    info, todo

    note

    参见：

    • pymdown-extensions/_admonition.scss at master · facelessuser/pymdown-extensions
    • reStructuredText Directives § Specific Admonitions
    • Admonition — Python-Markdown documentation
